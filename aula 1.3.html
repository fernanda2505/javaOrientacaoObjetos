As Plataformas JAVA e como elas evoluem

JAVA EE (JEE)

Plataforma mais completa. Vem junto de outras especificações.

SERVLETS e JSP(java server pages)

códigos para deixar no servidor, inclusive pode gerar o HTML para fazer algum sistema atraves do browser.

JSP: extensão para ficar mais fácil em usar o sistema.

JSF: JAVA SERVER FACES

Cria-se um compomente, um bloquinho e ele tem várias informações armazenadas para facilitar a interação do desenvolvedor.

JPA: java Persistene e API

Persistence: salvar seu banco de dados. (SQL POR EXEMPLO)

xJB: enterprise java beans

Fazer o usuário conversar na rede

JAVA ME: java micro edition

Lustração de demotração de uma logo, por exemplo

JAva card, java FX, javaTV

card: escrever programa para chip de celular, cartão de banco, para salvar contato, numero de telefone.

X: aplicações interativa, animações. Pode rodar na WEB ou no desktop.

Java tv: roda em tv digital.

Cresceu muito na Europa.

Como java evolui?

A população ajuda nisso. A JCP (java community process) ajuda a evoluir através das bibliotecas. Todos os dias surgem ideias e são implementadas diretamente no JAVA de acordo com a necessidade e curiosidade das pessoas. a JSR serve para especificar algo que você queira ou tenha por interesse. Ajuda inclusive a melhorar a performance
