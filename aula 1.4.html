Máquina virtual java (JVM)

O bytecode é o código intermediário entre o código fonte, o texto que o programador manipula e o código do computador que consegue executar. É como se fosse um mediador/tradutor entre o código o programador e a máquina.

>O JVM serve para facilitar a integração de comunicação entre o usuário e a máquina.

O objetivo é pegar um código e interpretar de acordo com o sistema operacional usado. A JVM não deixa o sistema lento, ele melhora o código e ele executa no JUST IN TIME (JIT) ou tempo real.
